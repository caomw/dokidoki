Detailed Description

Character is a GameObject, represents a character in game, used to take a series actions to tell game story. Those actions contain: RoleAction, PostureAction, TextAction, VoiceAction, MoveAction. Character could show appearence, speak voice, show psychological descriptions that are texts without voice, move around. Character GameObject itseft is a child of World GameObject Character has childs as Bubble board.
